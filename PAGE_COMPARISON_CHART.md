## ✅ QUALITY CHECKLIST

Before considering a page complete:

**Content Quality**
- [ ] Benefits-focused (not feature dumps)
- [ ] Clear value prop
- [ ] Addresses customer pain points
- [ ] Mobile-optimized copy
- [ ] Consistent brand voice

**Design & UX**
- [ ] Visual hierarchy clear
- [ ] CTA buttons prominent
- [ ] Scannable layout
- [ ] Whitespace utilized
- [ ] Mobile responsive

**Functionality**
- [ ] All forms working
- [ ] Links functional
- [ ] No 404s
- [ ] Analytics tracked
- [ ] Forms have confirmations

**Performance**
- [ ] Page load <3s
- [ ] Lighthouse 90+
- [ ] Images optimized
- [ ] No unused code
- [ ] Mobile fast

**Accessibility**
- [ ] WCAG AA compliant
- [ ] Color contrast OK
- [ ] Keyboard navigable
- [ ] Alt text on images
- [ ] Screen reader tested
